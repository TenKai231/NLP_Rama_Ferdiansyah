# 📱 Sentiment Analysis Review Aplikasi Shopee

Project ini merupakan implementasi **Natural Language Processing (NLP)** untuk melakukan analisis sentimen terhadap ulasan aplikasi Shopee menggunakan machine learning dan TensorFlow.

Model dapat mengklasifikasikan review menjadi:
- 😊 Positif
- 😐 Netral
- 😡 Negatif

---

# 🚀 Teknologi yang Digunakan

- Python
- TensorFlow / Keras
- Scikit-Learn
- TF-IDF Vectorizer
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Play Scraper
